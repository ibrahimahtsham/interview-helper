# Interview Helper

A cross-platform desktop app to assist with interview practice, built using PyQt5.

## Features

- Desktop UI (PyQt5)
- Easy to extend for audio, transcription, and LLM integration

## Getting Started

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Run the app:
   ```bash
   bash run.sh
   ```

## Structure

- `main.py`: Entry point for the app
- `ui/`: UI components
- `core/`: Core logic
- `README.md`: Project info
- `requirements.txt`: Python dependencies
- `run.sh`: Run script
- `.gitignore`: Git ignore file
